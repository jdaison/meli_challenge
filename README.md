# meli_challenge
Solution for Operation fuego quasar using typescript

## Installation
Clone or download this repository then run
```bash
npm i 
``` 

## Run
```bash
npm run start
``` 
## Run for dev
```bash
npm run dev
``` 

## Run unit tests
```bash
npm run test
``` 

## Swagger documentation
start the app and then open next url in your browser http://localhost:8080/docs/
